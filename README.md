# Angular6Template
A template for ASP.NET Core with Angular 6 

Install the template by running:
```shell
  dotnet new --install AspNetCore.Angular6::*
```

then simly run: 

```shell
  dotnet new angular6
```

Bear in mid to restore required packages:

```shell
  dotnet restore  
```
```shell
  cd ClientApp
  npm install
```

Run the project:

```shell
  dotnet run
```
